Tasks tracker

This is api that helps users to handle their tasks and meetings

Pjoject uses: 
    -python 3.10
    -Django 4.0.1
    -Postgres 15.1


Installation: 
    -open project
    -create virtual environment
    -install dependencies pip install -r requirements.txt
    -create .env file and add values: DB_ENGINE
                                        POSTGRES_DB
                                        POSTGRES_USER
                                        POSTGRES_PASSWORD
